# qrcode-3d-print

Generate autosized 3D qrcode printable file.

# Fonctionality

- Duplication
- Randomisation with UUID

# Export

- STL Exportable with seperates files per geometry for bicolor 3D printing
- STL Exportable with global file geometry

# Dev use

```
npm install
npm run dev
```
